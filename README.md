# 日田なび (Hita City Web Map)

## 概要
大分県日田市の観光・防災に関する総合案内コンシェルジュWebマップアプリケーションです。
GitHub Pagesで動作する静的なWebサイトとして構築されています。

## 特徴
- **観光モード**: 日田市内の観光スポット、イベント情報を地図上で確認。待ち時間を考慮した最適化ルートの算出。
- **防災モード**: 避難所、ハザードマップ情報の確認。最短距離での避難ルートの算出。
- **AI連携**: Gemini APIを使用したパーソナライズされた観光プランの提案。
- **GPS連携**: ブラウザのGeolocation APIを使用した現在地の取得。

## 使用方法
1. ブラウザで `index.html` を開く、またはGitHub PagesにデプロイしたURLにアクセスします。
2. 左上の「観光モード」「防災モード」を切り替えて利用します。
3. スポットを複数選択し、「最適化ルート算出」を行うと訪問順序が表示されます。

## 開発・構成
- **フロントエンド**: HTML5, CSS3, JavaScript (Bootstrap 5, Leaflet.js)
- **データ**: `spots.json` (Excelから変換されたスポットデータ)
- **レガシーコード**: Streamlit版のソースコードは `legacy/` フォルダに保管されています。
